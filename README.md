PKM-checksum-fixer
==================

Fixes the checksum of a .pkm file in case you've edit it. (like changing the O.T.)

Compilation
===========

Use the provided Makefile

How to use
==========

```bash
$ pkmcs Pikachu.pkm
```

You can fix several files in one go
```bash
$ pkmcs Pikachu.pkm Squirtle.pkm Charizard.pkm
```
